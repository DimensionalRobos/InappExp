Dependencies:
WordNet 2.1
JDK 1.8+

P.S. All Libs are at InappExpAPI/lib folder

Configuration:
Open the propsconfig file and change PythonFolder property to the folderpath to where the scripts folder is. Please make sure the path ends with backslash
For WordNetPath change it to "C:\Program Files\WordNet\2.1\dict" for 32-bit. For 64-bit change it to "C:\Program Files (X86)\WordNet\2.1\dict"

How to run:
Double Click InappExp.jar
or "java -jar InappExp.jar" from cmd
